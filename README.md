# B2B Demand Generation Dashboard

An interactive dashboard presenting a complete demand generation plan for an enterprise B2B SaaS AI product. Built as a portfolio case study demonstrating how to structure and present a multi-channel demand gen strategy for VP/CMO review.

**[View the live dashboard](https://neeldas1209.github.io/B2B-Demand-Gen-Dashboard/)**

## What's Inside

The dashboard covers the full demand gen planning process across 11 interactive tabs:

| Tab | What It Covers |
|-----|---------------|
| Market Context | Market landscape, category dynamics, and opportunity sizing |
| Competitive Landscape | Competitive positioning with radar chart comparison |
| ICP + Buying Committee | Ideal customer profile definition and buying committee mapping |
| Account Intelligence | Account research and intelligence gathering approach |
| Scoring + Tiering | Account scoring methodology and tier-based prioritization |
| Messaging Framework | Persona-specific messaging with value propositions |
| Programs + Channels | Multi-channel campaign strategy across paid, owned, and earned |
| Funnel Math + Attribution | Pipeline projections, conversion assumptions, and attribution model |
| 30/60/90/120 Timeline | Phased execution roadmap with milestones |
| AI Execution Layer | AI-powered workflows integrated into the demand gen process |

## Tech Stack

- Vanilla HTML, CSS, and JavaScript — single self-contained file
- Chart.js for data visualizations
- Plus Jakarta Sans typography
- No server, no build tools, no dependencies — opens in any browser

## Context

This was built as a case study for an enterprise B2B SaaS AI product launch. It demonstrates the ability to take a product from market analysis through competitive positioning, ICP definition, messaging, channel strategy, funnel math, and phased execution — all presented in an interactive format a marketing leadership team can explore and act on.
